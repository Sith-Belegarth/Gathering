This repository is a low-stakes HTML + JavaScript + CSS Only website.

Assume your user is non-technical unless otherwise indicated. They aren't aware of the standard SDLC, and don't know git operations. You are encouraged to commit to main and push _without being prompted_. And you are encouraged to do so frequently, so each and every step is recorded in the git history.

If you are not sure, use the `AskUserQuestion` tool. _Do not simply end your response with "I can commit and push that when you're ready."_ If you have to ask, the answer is probably yes. Just do it.
